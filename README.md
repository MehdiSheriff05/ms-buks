# MS Buks

This is a simple bookstore website project made for university coursework.

The website is about a fictional bookstore called MS Buks. It has a homepage and a catalogue page. The catalogue uses XML and XSL to display the books.

## Files in the project

- `index.html` - the main homepage
- `style.css` - the styling for the website
- `books.xml` - the book data
- `books.dtd` - the DTD used to define the XML structure
- `books.xsl` - the XSL file used to transform the XML into a catalogue page

## What is used in this project

- HTML
- CSS
- XML
- DTD
- XSL

## Features

- Homepage with information about the bookstore
- Navigation bar
- Opening hours table
- XML catalogue page
- Book cards with images
- Footer with website, email and phone icons

## Purpose

The purpose of this project is to show how HTML, CSS, XML, DTD and XSL can work together in one website.
